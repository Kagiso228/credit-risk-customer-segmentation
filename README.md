# Credit Risk & Customer Segmentation

##  Overview
This project focuses on predicting credit card default risk and segmenting customers based on behavioural patterns. The goal is to support better financial decision-making through data-driven insights.

## ⚙️ Tech Stack
- Python (Pandas, NumPy, Scikit-learn)
- Machine Learning
- Power BI
- SQL (optional)

##  Key Features
- Feature engineering (credit utilisation, payment behaviour, delays)
- Predictive modelling for default risk
- Customer segmentation using clustering
- Interactive dashboard for business insights

##  Results
- ROC-AUC: 0.68
- Recall (default class): 54%
- Improved detection of high-risk customers in an imbalanced dataset
